# Chur Abfallplan

Home Assistant Integration fuer die Abfalltermine der Stadt Chur.

## Installation ueber HACS

1. HACS oeffnen.
2. Benutzerdefiniertes Repository hinzufuegen: `https://github.com/MasterBals/Chur-Abfallplan`.
3. Kategorie `Integration` waehlen.
4. `Chur Abfallplan` installieren.
5. Home Assistant neu starten und die Integration unter Geraete & Dienste einrichten.

## Quelle

Die Integration nutzt oeffentlich abrufbare Informationen der offiziellen Chur-Webseiten.
